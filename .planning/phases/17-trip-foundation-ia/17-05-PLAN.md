## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-17-13 | Tampering | trip create city/date input | mitigate | Payload validated through `TripCreateSchema` (city_code max 20, dates required, end>=start refine) before `createTrip`; DB CHECK constraints (city_code length) back it server-side (V5 Input Validation, ASVS L1). |

</threat_model>
