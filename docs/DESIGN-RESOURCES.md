# 디자인 리소스 모음

> 작업하면서 아이콘·이미지·일러스트가 필요할 때 바로 참고하는 링크 모음.
> MOAJOA 스택(Web: Next.js 15 / iOS: Expo) 기준으로 바로 쓸 수 있는 것들 위주.

---

## 아이콘 (Icon)

| 이름 | 링크 | 설명 |
|---|---|---|
| **Lucide** | https://lucide.dev | 깔끔하고 일관성 있는 오픈소스 아이콘 1,500+. React/Vue/Svelte 패키지 제공해서 개발에 바로 쓰기 편함. (Feather Icons 후속) |
| **Heroicons** | https://heroicons.com | Tailwind CSS 팀 제작. MIT 라이선스(상업적 자유). outline/solid 두 스타일. |
| **Phosphor Icons** | https://phosphoricons.com | 9,000+ 아이콘 × 6가지 굵기(weight). 선택지가 매우 넓음. |
| **Tabler Icons** | https://tabler.io/icons | 5,000+ 무료 SVG. MIT 라이선스. |
| **Material Symbols** (구 Material Icons) | https://fonts.google.com/icons | 구글의 방대한 아이콘 세트. 안드로이드/머티리얼 디자인 앱에 잘 어울림. |
| **Font Awesome** | https://fontawesome.com | 가장 유명한 아이콘 라이브러리 중 하나. Free tier도 충분히 많음. |
| **Iconify** | https://iconify.design · 검색: https://icon-sets.iconify.design | 위 여러 세트(20만+)를 한 곳에서 검색/사용. 라이브러리 비교할 때 유용. |

> 💡 **MOAJOA 추천**: Lucide. React/Expo 양쪽 패키지(`lucide-react`, `lucide-react-native`)가 있어 Web/iOS 일관성 유지에 좋음.

## 이미지 / 사진 (Photo)

| 이름 | 링크 | 설명 |
|---|---|---|
| **Unsplash** | https://unsplash.com | 고품질 무료 사진의 대표. 상업적 사용 가능 + API 제공. |
| **Pexels** | https://pexels.com | 사진 + 동영상 모두 제공. 무료 상업적 사용 가능. |
| **Pixabay** | https://pixabay.com | 사진·일러스트·벡터·동영상까지 폭넓게 제공. |
