## Typography

정확히 4 사이즈 / 2 웨이트. 전부 기존 `typography` 토큰:

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Caption | 12px (`text-xs`) | 400 | 1.5 | 스텝 카운터, 이동시간 캡션, 보조 문구(D-25), 진행 표시 부가 텍스트, 배지 숫자(600 예외) |
| Label | 14px (`text-sm`) | 400 | 1.5 | 서브카피(중앙 회색), 주소, 섹션 카운트("아직 안 넣은 곳 3"), 보조 버튼(sm) |
| Body | 16px (`text-base`) | 400 / **600** | 1.5 | SelectPill 라벨(600), 장소명(600), CTA(600), 입력 필드(400) |
| Display | **24px** (`text-2xl`) | 600 | 1.25 (`leading-tight`) | 위저드 스텝 타이틀(중앙, 최대 2줄) · [일정] 섹션 타이틀 |

**Weights: 400(regular) + 600(semibold)만.** 레퍼런스의 700(bold) 타이틀은 **토큰 semibold 600으로 매핑**(신규 웨이트 도입 금지).

**Exceptions (pre-existing — 무수정, Karpathy §3.3):**
- 18px `text-lg` — 시트 헤더(모아 이름)·빈 상태 heading. Phase 24가 선언한 Heading 롤, 이 phase는 손대지 않음.
- `Chip`의 12px/500 — 기존 컴포넌트. 신규 코드에서 weight 500 사용 금지.

| Role | Token | Value | Usage |
|------|-------|-------|-------|

| Accent (10%) | `brand.500` / `brand.600` | #2979FF / #2563EB | 아래 reserved 목록만 |
| Accent(약) | `brand.300` | #93B4FF | **비활성 CTA 채움 전용** (레퍼런스 "연한 파랑 bg + 흰 글씨") |
| Destructive | `semantic.danger` | #EF4444 | 추출 실패 행(기존) — **이 phase에 파괴적 액션 없음** |
| Text | `neutral.900` / `neutral.600` / `neutral.500` | — | 본문 / 비선택 pill 텍스트 / 서브카피·메타 |

**Accent reserved for (완전 목록 — 이 외 사용 금지):**
1. Primary CTA 버튼 (`brand-600` — 기존 Button primary)
2. **비활성 CTA 채움** (`brand-300` + 흰 글씨 — D-05)
3. **선택된 SelectPill**: 흰 배경 + `border-brand-500` 테두리 + `text-brand-600` 텍스트 (D-04)
4. **스텝 카운터 `N/4`** (`text-brand-600`)
5. **선택된 Day 탭** (`bg-brand-500` + 흰 텍스트) / 비선택 = 흰 배경 + `border-neutral-300`
6. **타임라인 번호 배지** (`bg-brand-500` + 흰 숫자) 및 **지도의 Day 번호 핀** (같은 값 — 지도↔리스트 동일 번호 대응, D-16)
7. **[일정] 섹션 타이틀의 '추천 일정' 어절** (`text-brand-600` — 레퍼런스 IMG_2922 정체성)
8. 선택된 이동수단 세그먼트 (`bg-white` + `text-brand-600`, 트랙은 `neutral-100`)
9. (기존 Phase 24 승계) FAB `brand-600` · 호스트 핀/순번 배지 `brand-500` · 찜 하트 filled `brand-500` · 캘린더 range `brand-600`/`brand-100` · 선택된 공유 모드 카드

**대비 근거 (WCAG 2.1 AA):**
- 선택 pill 텍스트는 **`brand-600`**(흰 배경 대비 5.17:1 — PASS). `brand-500`(3.72:1)은 16px 텍스트에 미달이므로 **테두리·배지 등 비텍스트에만** 사용(1.4.11 비텍스트 대비 3:1 충족).
- 비선택 pill: `neutral-600` on `neutral-100` = 6.88:1 (PASS).
- **비활성 CTA(`brand-300` + 흰 글씨, 2.06:1)는 disabled 컨트롤 — WCAG 1.4.3 대비 요건 면제 대상**이며 D-05가 명시적으로 잠근 시각이다. 활성/비활성 구분은 색만이 아니라 `disabled` 속성 + `aria-disabled`로도 전달된다.
