## Spacing Scale

Declared values (multiples of 4). 기존 `spacing` 토큰(ui-tokens)의 부분집합만 사용:

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | 아이콘-텍스트 간격, 배지 내부 |
| sm | 8px | 칩 간 간격, 행 내부 요소 간격 |
| md | 16px | 기본 요소 간격, 화면 좌우 패딩(모바일), 리스트 행 상하 패딩 |
| lg | 24px | 시트 좌우 패딩(px-6 — 기존 BottomSheet 계승), 섹션 간격 |
| xl | 32px | 온보딩 스텝 헤딩↔콘텐츠 간격 |
| 2xl | 48px | 온보딩 상단 여백(스텝 인디케이터 아래) |

**Exceptions:**
- 터치 타깃 최소 **44×44px** — 모든 탭 가능한 행·버튼·마커·FAB·칩(칩은 시각 높이는 작아도 hit area 44px 확보)
- 기존 Chip 컴포넌트 py-1.5(6px) — pre-existing, 무수정 (Karpathy §3.3)
- 드래그 핸들 h-1.5 w-10 — 기존 BottomSheet 시각 언어 그대로

## Typography

정확히 4 사이즈 / 2 웨이트. 전부 기존 `typography` 토큰의 부분집합:

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Caption | 12px (`text-xs`) | 400 | 1.5 | 순번 배지 숫자(600 예외 아래), "닉네임님이 담음", 타임스탬프, 스텝 안내 문구 |
| Label | 14px (`text-sm`) | 400 | 1.5 | 행 메타(주소·출처), 시트 카드 설명 한 줄, 보조 버튼(sm) |
| Body | 16px (`text-base`) | 400 | 1.5 | 장소명(600), 입력 필드, 기본 버튼, 모아 카드 제목(600) |
| Heading | 18px (`text-lg`) | 600 | 1.25 (tight) | 온보딩 스텝 질문, 시트 타이틀(기존 BottomSheet title과 동일), 지도탭 상단 바 타이틀 |

**Weights: 400(regular) + 600(semibold)만.** 강조는 사이즈가 아니라 600으로.
**Exceptions:** 기존 Chip 컴포넌트의 `font-medium`(500)과 순번 배지 숫자 600@12px — pre-existing 시각 언어 유지. 신규 코드에서 500 사용 금지.

## Color

전부 `@moajoa/ui-tokens` `colors` 참조 — 신규 hex 리터럴을 컴포넌트에 직접 쓰지 말 것.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `surface.background` #F5F6F8 | 앱 스캐폴드 — 온보딩·/moa 리스트 배경. 지도탭에서는 지도 자체가 dominant |
| Secondary (30%) | `surface.raised` #FFFFFF | 시트·카드·상단 바·입력 필드 — white-only raised 표면 (기존 디자인 시스템 규칙) |
| Accent (10%) | `brand.500` #2979FF / `brand.600` #2563EB | 아래 reserved 목록만 |
| Destructive | `semantic.danger` #EF4444 | 추출 실패 행 아이콘·문구 강조만 (이 phase에 파괴적 액션 없음) |
| Text | `neutral.900` 본문 / `neutral.500` 보조 | 기존 관례 |

**Accent reserved for (완전 목록 — 이 외 사용 금지):**
1. Primary CTA 버튼 (`brand-600` — 기존 Button primary variant)
2. 선택된 칩 (`brand-100` 배경 + `brand-700` 텍스트 — 기존 Chip selected)
3. 호스트 핀·순번 배지 (`brand-500` — D-20 고정)
4. FAB [+] 버튼 (`brand-600` + `shadows.fab`)
5. 함께 정하기 시트의 선택된 모드 카드 (border `brand-500` + `brand-50` 배경)
6. 찜 하트 filled 상태 (`brand-500` — pin.loved 시맨틱 추종, 기존 토큰 주석 "loved tracks brand-500")
7. 캘린더 range 선택 (endpoints `brand-600` 원형, 사이 날짜 `brand-100` 띠)
8. 스텝 인디케이터 현재 dot (`brand-600`)

## Screen Contracts

### 레이아웃 공통 (D-10)
- 모바일 퍼스트, 단일 구현. 데스크톱 = **max-w-lg(512px) 중앙 고정** — 기존 BottomSheet max-w-lg와 폭 일치. 바깥 거터는 `neutral.100`.

### `/onboarding` (D-02 단일 라우트, 클라이언트 스텝)
- 배경 `surface.background`, 콘텐츠 좌우 16px.
- **스텝 인디케이터 [AUTO]:** 상단 중앙 4-dot — 현재 `brand-600` w-6 pill, 나머지 `neutral-300` w-1.5 원. 숫자 표기 없음.
- 상단 좌측: step 2~4에서 뒤로 chevron(lucide `chevron-left`, 44px 타깃) — 스텝 후퇴.
- 헤딩 18/600, 아래 32px 뒤 콘텐츠.
- **전환 애니메이션 [AUTO]:** 없음(즉시 교체). 과설계 금지 — 시트 250ms 모션만이 이 phase의 모션 예산.
- CTA는 하단 고정(full-width primary md, 좌우 16px, safe-area 하단 패딩). 선택 전 disabled.
- step 2: "날짜 정했어요"/"아직 미정이에요" 2택 카드(선택 시 border brand-500) — 확정 선택 시 카드 아래 캘린더 노출, 미정 선택 시 안내 한 줄(14/400 neutral-500) 노출 후 CTA 활성.
- step 4: 탭 2개(링크/검색) + 담은 항목 스테이징 리스트(로컬 상태 — D-03: 생성은 완료 시 일괄). CTA "모아 만들기", 그 위 "건너뛰기" text button 중앙.

### `/moa/[id]` 지도탭 (D-09)
- **지도:** 풀 컬럼·풀 하이트, greedy gesture + clickableIcons false (public-board-map 옵션 계승). 마커 = `buildMarkerIconUrl` fill을 추가자 색으로.
- **상단 바:** 지도 위 오버레이 — 좌 back chevron(흰 원형 40px + shadow.sm), 중앙 없음, 우 **[함께 정하기]** primary sm 버튼. 오버레이 요소만, 불투명 바 금지(지도 감성 유지).
- **FAB [+]:** `brand-600` 56px 원형 + `shadows.fab`, lucide `plus` 흰색. 우측 하단, collapsed 시트 상단에서 16px 위. z-order: 시트보다 위(expanded에서도 접근 가능).
- **place-sheet:** §Interaction 참조. 시트 헤더 = 모아 이름(18/600) + "장소 {N}개"(14/400 neutral-500).
- **리스트 행 anatomy** (상하 16px, 좌우 24px, 최소 높이 44px):
  - 좌: **순번 배지** — 20px 원, fill=추가자 핀 색, 흰 12px/600 숫자 = `place.seq_no` 불변 (MOA-02: 정렬 바뀌어도 숫자 그대로)
  - 중: 장소명 16/600 neutral-900 (1줄 truncate) + 아래 "{닉네임}님이 담음" 12/400 neutral-500
  - 우: **찜 하트 토글 [AUTO — RESEARCH Open Q2 권고안 채택: 호스트 찜 포함]** — lucide `heart`, off=neutral-400 outline / on=`brand-500` filled, 옆 count 12/400. optimistic+reconcile
- **정렬:** 찜 desc → seq_no asc (MOA-02, `sortByLove` comparator).
- **분석 중 행 (D-13):** spinner(브랜드 아님 — neutral-400) + 링크 title/URL 14/400 truncate + "분석 중…" 12/400 neutral-500.

- **아코디언 (MOA-05) — 정보 순서 [AUTO]:**
  1. 주소 — 14/400 neutral-700
  2. 액션 행(가로, 8px gap): "구글맵에서 보기" text sm + external-link 아이콘 → `buildGoogleMapsPlaceUrl`

  - 펼침/접힘: max-height transition 200ms ease-out. 한 번에 하나만 펼침.
- **Empty state** (장소 0): 시트 안 — heading 18/600 + body 14/400 neutral-500, 중앙 정렬, FAB 방향 안내.
