1. 내 애플리케이션 → 애플리케이션 추가 (이미 있으면 재사용)
2. 앱 키 → **REST API 키** 복사 (= client_id)
3. 제품 설정 → 카카오 로그인 → **활성화 ON**

**B. Supabase Dashboard** (https://supabase.com/dashboard → 프로젝트 → Authentication):
1. **Anonymous sign-ins 토글 ON** (Sign In / Providers 설정)
2. Providers → **Kakao 활성화** → A-2의 REST API 키를 Client ID에, A-4의 Secret 입력 → 저장

**C. 로컬 실값 배선 (선택 — 로컬에서 실제 카카오 로그인 테스트하려면):**
1. `supabase/.env.local`에 `KAKAO_REST_API_KEY=<실값>` / `KAKAO_CLIENT_SECRET=<실값>` 추가 (이 파일은 gitignored — 커밋 금지)
2. 실값을 1Password/노션 비밀 페이지에 보관 (§4.7)

**확인 방법:** B 완료 후 브라우저에서 `https://<project-ref>.supabase.co/auth/v1/authorize?provider=kakao` 접속 → kauth.kakao.com 로그인 화면으로 redirect되면 성공 (성공 기준 4 — "OAuth 플로우 시작 가능". 콜백 완주·버튼 UI는 Phase 24).
