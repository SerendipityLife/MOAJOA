### T2 — Marker SVG icon 분기 (`public-board-map.tsx`)

**File:** `apps/web/app/b/[slug]/_components/public-board-map.tsx`

기존 `new g.Marker({ map, position, title })` 호출(lines 50-54)을 SVG icon 포함 형태로:

```ts
import { LOW_CONFIDENCE_THRESHOLD } from '@moajoa/core';

for (const p of places) {
  const isAi = p.source_kind === 'ai';
  // confidence may be undefined in older cached RPC payloads — treat as high conf
  const conf = (p as { confidence?: number | null }).confidence;
  const isLowConf = isAi && conf !== null && conf !== undefined && conf < LOW_CONFIDENCE_THRESHOLD;
  const fill = isAi ? '#F97316' : '#0F172A';
  const fillOpacity = isLowConf ? 0.45 : 1.0;
  const showQ = isLowConf;

  const svg = `<svg xmlns="http://www.w3.org/2000/svg" width="32" height="40" viewBox="0 0 32 40">
    <path d="M16 0C7.16 0 0 7.16 0 16c0 9.6 16 24 16 24s16-14.4 16-24C32 7.16 24.84 0 16 0z" fill="${fill}" fill-opacity="${fillOpacity}"/>
    ${showQ ? '<text x="16" y="22" text-anchor="middle" font-size="14" font-family="sans-serif" fill="#ffffff">?</text>' : ''}
  </svg>`;

  const marker = new g.Marker({
    map,
    position: { lat: p.lat, lng: p.lng },
    title: p.name_local,
    icon: {
      url: `data:image/svg+xml;utf-8,${encodeURIComponent(svg)}`,
      scaledSize: new g.Size(32, 40),
      anchor: new g.Point(16, 40),
    },
  });

  // Pin click → YouTube — UNCHANGED from Phase 4 D-14.
  if (p.link_id) {
    const link = linksById.get(p.link_id);
    if (link?.url) {
      const youtubeUrl = buildYouTubeWatchUrl(link.url, p.source_timestamp_sec);
      if (youtubeUrl) {
        marker.addListener('click', () => {
          window.open(youtubeUrl, '_blank', 'noopener,noreferrer');
        });
      }
    }
  }
}
```

> **`confidence` undefined 방어:** RPC jsonb에 confidence가 0006 마이그 후 들어오지만, Vercel edge cache에 stale revision이 있을 수 있음. `undefined` → falsy → high conf 처리(opacity 1.0) — 안전 fallback.

**Verify:** Mobile Safari (iPhone) 시뮬레이터에서도 SVG marker 렌더 정상. 클릭 핸들러 동작 그대로.

## Risk / Pitfalls

- **SVG `?` 글자 font-family:** Google Maps SVG icon은 user system font에 의존. sans-serif fallback 안전.
- **`scaledSize`/`anchor` 명시 안 하면 marker 위치 어긋남:** anchor (16, 40) = SVG의 핀 끝 좌표.
- **Stale Vercel ISR:** revalidateTag 후 cache miss 발생할 때만 새 RPC payload. Phase 4 `/api/revalidate` webhook이 처리 (회귀 없음).
- **`AdvancedMarkerElement` 대신 `Marker` 사용:** Phase 4가 이미 legacy `Marker` 사용 중. 변경 X (외과적).
